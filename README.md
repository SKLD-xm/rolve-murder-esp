# rolve-murder-esp
A working ESP for the ROLVe game Murder

# Features
> Role colors
  - Red: Murderer
  - Blue: Bystander
  - Yellow: Bystander with weapon
  - Dark red: Frozen person (Cold Killer Only)
> Weapon colors
  - Magenta: Revolver
  - White: Knife (if thrown)
> Loot colors
  - Green: All loot
> Player's respective names
